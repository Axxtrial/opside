Introducción

Opside Bot es una potente herramienta que permite administrar tu servidor Minecraft directamente desde Discord. Esta documentación te guiará a través de todas las funcionalidades y características disponibles.

Requisitos

  • Servidor Discord con permisos de administrador
  • Servidor Minecraft con RConsole habilitado
  • Acceso al panel de control del servidor

Comandos Básicos

Estos son los comandos básicos disponibles para todos los usuarios:

Comando Descripción Uso
/status Muestra el estado del servidor /status
/list Lista los jugadores conectados /list
/tps Muestra el TPS del servidor /tps

Comandos de Moderación

Comandos para la gestión de jugadores y moderación del servidor:

Comando Descripción Uso
/ban Banea a un jugador /ban [jugador] [razón]
/unban Desbanea a un jugador /unban [jugador]
/kick Expulsa a un jugador /kick [jugador] [razón]
/warn Envía una advertencia /warn [jugador] [razón]

Comandos Personalizados

Crea tus propios comandos personalizados para automatizar tareas:

Crear un comando personalizado

/opside custom add "nombre" "comando"

Ejemplos

/opside custom add "startevent" "startevent %event%"
/opside custom add "stopevent" "stopevent %event%"

Configuración

Configura el bot según tus necesidades:

Configuración básica

/opside setup

Configuración de canales

/opside channel set #canal-comandos
/opside channel set #canal-logs

Sistema de Permisos

Gestiona quién puede usar qué comandos:

Agregar permisos

/opside permissions add @rol

Remover permisos

/opside permissions remove @rol

Ver permisos

/opside permissions list

Sistema de Logs

Configura y gestiona los logs del servidor:

Configurar canal de logs

/opside logs channel #canal-logs

Ver logs recientes

/opside logs recent [cantidad]
