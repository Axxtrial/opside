Guía de Instalación

1. Requisitos Previos

Antes de comenzar, asegúrate de tener:

  • Un servidor Discord con permisos de administrador
  • Un servidor Minecraft con RConsole habilitado
  • Acceso al panel de control de tu servidor

2. Invitar el Bot a tu Servidor

Para agregar Opside Bot a tu servidor:

  1. Haz clic en el botón "Agregar a Discord" en la página principal
  2. Selecciona tu servidor de la lista
  3. Autoriza los permisos necesarios

3. Configurar RConsole

Para conectar el bot con tu servidor Minecraft:

  1. Abre el archivo server.properties de tu servidor
  2. Habilita RConsole agregando estas líneas:
    3. enable-rcon=true
rcon.password=tu_contraseña_segura
rcon.port=25575
  3. Reinicia tu servidor Minecraft

4. Configurar el Bot

Una vez que el bot esté en tu servidor:

  1. Usa el comando /setup para iniciar la configuración
  2. Ingresa la dirección IP de tu servidor
  3. Ingresa el puerto RConsole (por defecto 25575)
  4. Ingresa la contraseña RConsole que configuraste

5. Verificar la Instalación

Para asegurarte de que todo funciona correctamente:

  1. Usa el comando /status para verificar la conexión
  2. Prueba un comando básico como /list
  3. Verifica que los permisos estén correctamente configurados

6. Solución de Problemas

Si encuentras algún problema:

  • Verifica que RConsole esté habilitado y funcionando
  • Asegúrate de que el bot tenga los permisos necesarios en Discord
  • Comprueba que la contraseña RConsole sea correcta
  • Verifica que el puerto RConsole no esté bloqueado por el firewall
